Wähle deinen Modus

Von schnellen Duellen bis zum Endlos‑Run – für jeden Moment der richtige Modus.

6 Runden, 3 Fragen pro Runde (18 gesamt) gegen zufällige Gegner.

  • Zufallsgegner
  • Schnell

Freunde

  • 18 Fragen
  • Freunde einladen

  2. 2

    Run starten

    Schnell rein, keine lange Vorbereitung. Gegner wird automatisch gefunden.

    Fragen beantworten

    Punkte sammeln, Ranking klettern und den nächsten Run starten.
